LISTAS

Generalmente se usa las listas ordenadas y desordenadas, pero existe una tercera.

1) Listas ordenadas: La etiqueta es ol de ordered list. Ahi dentro cada elementos tiene que ir con una etiqueta de li, por list order

  1. Primavera
  2. Verano
  3. Otoño
  4. Invierno
Puedo crear una lista y decirle en que numero empezar con la caracteristica start:
  1. Dos
  2. Tres
  3. Cuatro
Puedo hacerla en el orden inverso con el atributo reversed. Es un valor booleano, por lo que aunqe necesita ser true, por el hecho de estar significa q es true, le podria poner false y entonces no seria reversed:
  1. Tres
  2. Dos
  3. Uno
Nos permite cambiar de numeros a letras, enumeracion romana y otros con el atributo type:
  1. jamon
  2. huevo
  3. mayonea

Listas desordenadas: Es cuando hacemos listas con viñetas o bullets. La etiqueta es ul por unordered list

  • lavar
  • ordenar
  • arreglar el estante
Aca tambien se puede usar el atributo type para cambiarle la enumeracion:
  • jugar
  • dormir
  • descansar
  • jugar
  • dormir
  • descansar

Listas por definicion: no son muy utilizadas, sirven para cosas puntuales como glosario de terminos. Se forman con la etiqueta dl de definition list. Dentro de la lista esta el dt, q seria el termino y luego el dd que seria la definicion:

HTML
Es un lenguaje de marcado q define el contenido de la web
CSS
Es un lenguaje de definicion que da estilos al codigo html
JavaScript
Es el lenguaje de la web
